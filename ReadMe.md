
This project reveals electronic white lining from 811 "call before you dig" notices.
Which puts local Menlo Park homeowners on an even footing with facilities owners,
who can already view such electronic markings.

# quickstart

```bash
make install lint
```

Packages are managed by
[uv](https://docs.astral.sh/uv).
Testing happens on macos & linux.
Windows users might choose to use WSL2.
